### 书籍推荐
- 《JavaScript DOM 编程艺术》
被推荐最多的前端入门书籍

- 《JavsScript高级程序设计（第三版）》
学习 JS 最好的书，学习 JS 每个阶段再去读，仍然会有收获。

- [You-Dont-Know-JS](https://github.com/getify/You-Dont-Know-JS/tree/2nd-ed/get-started)
GitHub 上很火的 JS 教程，是英文版。中文是《你不知道的JavaScript》
可以用来提升自己对 JS 语法的理解程度。
